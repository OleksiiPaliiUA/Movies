- [Node](https://nodejs.org/en/) 16.18.0

## Configuring App

If you have all the prerequisites installed you should verify if your PostgreSQL is up.

```

$ cd movies
$ cd server
$ npm install
$ npm run seed (For creating tables/extensions!)
$ npm run dev (nodemon src/index.ts)

```
